### 🏠 Proyecto Destacado: Sistema de Gestión y Control de Pagos para la Urbanización Villa Heroica

Desarrollé este sistema como parte de mi servicio comunitario para resolver un problema crítico y real en Venezuela: la complejidad en la gestión de pagos de condominio y el manejo multimoneda (Bs/$).

**¿Qué hace destacar este sistema?**
* **Transparencia financiera:** Automatiza el cálculo de deudas sincronizándose con la tasa oficial del BCV del día.
* **Diseño centrado en el usuario:** Creé un panel de administración visual e intuitivo, pensado para usuarios sin experiencia técnica.
* **Seguridad:** Implementación de estándares actuales de encriptación (Argon2) y protección robusta contra vulnerabilidades web.
* **Arquitectura de datos:** Diseño de base de datos normalizada para garantizar la consistencia y escalabilidad del sistema.
