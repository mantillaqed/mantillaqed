### 🏠 Featured Project: Management and Payment Control System for Villa Heroica Urbanization

I developed this system as part of my community service to solve a critical, real-world problem in Venezuela: the complexity of managing condominium payments and multi-currency handling (Bs/$).

**Why does this system stand out?**
* **Financial transparency:** Automatically calculates debts by syncing with the official daily Central Bank of Venezuela (BCV) exchange rate.
* **User-centered design:** I created an intuitive, visual administrative panel designed for users without technical experience.
* **Security:** Implemented current encryption standards (Argon2) and robust protection against common web vulnerabilities.
* **Data architecture:** Database design based on normalization to ensure system consistency and scalability.
